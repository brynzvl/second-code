Semantic Elements

In HTML5, a developer can create divisions in the code by using the ‘div’ element.
However, in an effort to be more organized and clear, a developer can use semantic elements in their code by clearly labeling their div.
These elements use language that both humans and machines can understand.

Some examples of semantic elements include Header, Footer, Nav, Article, Form, Table, and Img.
